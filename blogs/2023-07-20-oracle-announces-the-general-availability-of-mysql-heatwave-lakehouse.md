Oracle today announced the general availability of MySQL HeatWave Lakehouse, delivering an industry first by enabling customers to query data in object storage as fast as querying data inside the database. MySQL HeatWave Lakehouse supports a variety of object store file formats such as CSV, Parquet, and export files from other databases, and can combine object storage file data and MySQL database transactional data together in the same query.
