Oracle today announced that MySQL HeatWave is available on Amazon Web Services (AWS).  MySQL HeatWave is the only service that combines OLTP, analytics, machine learning, and machine learning-based automation within a single MySQL database.  MySQL HeatWave delivers 7X better price performance compared to Amazon Redshift and 10X better than Snowflake, 25X faster than Redshift ML, and up to 10X higher throughput than Aurora.
