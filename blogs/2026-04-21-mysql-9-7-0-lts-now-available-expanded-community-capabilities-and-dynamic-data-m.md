Oracle today announced the availability of MySQL 9.7.0 LTS, establishing the new 9.7.x Long-Term Support release line. This release expands MySQL Community Edition with significant new capabilities across replication observability, high availability behavior, telemetry, JSON-relational development, and query optimization, while also introducing Dynamic Data Masking in MySQL Enterprise Edition.

Read the blog to learn more about what is new in MySQL 9.7.0 LTS and how this release reflects MySQL’s continued focus on delivering more innovation to the community.
